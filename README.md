# ☕ Coffee Shop Sales Analysis Dashboard

## Short Description
This project analyzes coffee shop sales data to understand sales performance, customer purchasing patterns, and product trends. The dashboard provides interactive insights that help identify popular products, peak sales times, and revenue distribution.

## Tech Stack
-  Excel / CSV Dataset

## Data Source
[Coffee shop sales dataset](https://mavenanalytics.io/data-playground/coffee-shop-sales)

## Features and Highlights
-  Interactive Power BI dashboard
-  KPI cards for total sales, orders, and quantity
-  Time-based analysis of sales trends
-  Product category performance analysis
-  Visual insights using charts and graphs
-  Filters and slicers for interactive exploration

## Example Insights
The dashboard helps analyze:
-  Best-selling coffee products
-  Peak sales days and hours
-  Popular product categories
-  Revenue contribution by product type

## Key Questions and Insights
- Which products generate the highest sales?
- What time of day has the most orders?
- Which product categories are most popular?
- How do sales trends change over time?
